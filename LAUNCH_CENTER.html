üîß Troubleshooting

‚ùå Port Already in Use

Problem: Ports 8080 or 8081 are already in use

Solution: Kill the process using the port or change the port in Main.java

netstat -ano | findstr :8080 taskkill /PID /F

‚ùå Database Connection Error

Problem: "Failed to initialize connection pool"

Solution: Ensure database/ folder exists and has write permissions

mkdir database

‚ùå WebSocket Connection Refused

Problem: Game client can't connect to server

Solution: Verify server is running and firewall allows port 8080

‚ùå Gradle Build Fails

Problem: Compilation errors when building

Solution: Ensure Java 11+ is installed and JAVA_HOME is set

java -version echo %JAVA_HOME%
